# Hotel Booking Data Analysis

## Overview
This project analyzes hotel booking data to identify patterns, cancellation behavior, and business insights.

## Files Included
- Jupyter Notebook (analysis code)
- Visualizations (charts)
- Summary Report (PDF/PPT)

## Key Insights
- ~20% bookings are cancelled
- Higher cancellations in Travel Agent bookings
- Standard rooms show higher cancellation rates
- Seasonal variation in booking values

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
